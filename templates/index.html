파이 검색과 다양한 수학 기능들

파이 검색과 다양한 수학 기능들을 제공합니다.
검색을 시작하거나 기능들을 살펴보세요.

기능 살펴보기

🔎 파이 검색

파이 속에서 원하는 숫자를 찾아보세요.

🔢 소수판별

원하는 숫자가 소수인지 합성수인지 확인해보세요.

🧮 소인수분해

원하는 숫자를 소인수분해 해보세요.

🔀 랜덤

범위 내에서 랜덤한 숫자를 뽑아보세요

🎲 주사위

원하는 주사위 눈 개수로 주사위를 굴려보세요.

⏫ 최대공약수

원하는 수의 최대공약수를 구해보세요.

⏬ 최소공배수

원하는 수의 최소공배수를 구해보세요.

🖥️ 디스코드 봇

디스코드에서 바로 파이를 검색하세요.

📦 오픈소스

프로젝트 코드를 확인하고 기여해보세요.
