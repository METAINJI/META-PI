개인정보 처리방침

최종 업데이트: 2026년 3월 27일

이 개인정보 처리방침은 META Pi("당사", "저희" 또는 "우리")가 당사의 Discord 봇 서비스("서비스")를 이용하실 때 정보를 수집, 사용 및 보호하는 방법을 설명합니다.

1. 수집하는 정보

귀하가 당사 서비스를 이용할 때, 당사는 다음과 같은 정보를 수집할 수 있습니다.

  • 디스코드 사용자 ID
  • 디스코드 사용자 이름/닉네임
  • 서버(길드) ID
  • 명령어 사용 데이터

저희는 실명, 주소, 결제 정보와 같은 민감한 개인 정보를 수집하지 않습니다.

2. 정보 활용 방식

수집된 데이터는 서비스 운영 및 개선 목적으로만 사용되며, 여기에는 다음이 포함됩니다.

  • 봇 기능 제공
  • 기능 구현을 위해 사용자 관련 데이터를 저장합니다.
  • 서비스 안정성 및 보안 유지

3. 데이터 저장

귀하의 데이터는 서버 데이터베이스와 같은 제3자 서비스에 저장될 수 있습니다.

저희는 고객님의 데이터를 보호하기 위해 합리적인 조치를 취하고 있지만, 절대적인 보안을 보장할 수는 없습니다.

4. 데이터 보존

  • 사용자 데이터는 봇 기능에 필요한 기간 동안만 저장됩니다.
  • 언제든지 삭제를 요청할 수 있습니다.

5. 데이터 공유

당사는 다음과 같은 경우를 제외하고 귀하의 개인 정보를 제3자에게 판매하거나 공유하지 않습니다.

  • 법률에 의해 요구되는 경우
  • 서비스 운영에 필요한 경우 (예: 호스팅, 데이터베이스)

6. 사용자 권한

당신은 다음과 같은 권리를 가지고 있습니다:

  • 데이터 접근 권한을 요청하세요
  • 본인의 데이터 수정 또는 삭제를 요청하세요.

7. 아동의 개인정보 보호

본 서비스는 만 13세 미만 사용자를 대상으로 하지 않습니다. 당사는 아동의 데이터를 고의로 수집하지 않습니다.

8. 본 정책의 변경 사항

당사는 언제든지 본 개인정보처리방침을 업데이트할 수 있습니다. 서비스를 계속 이용하는 것은 변경 사항에 동의하는 것으로 간주됩니다.
